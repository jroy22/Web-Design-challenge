Summary: Latitude vs. X

For this project, we sought to examine the effects of latitudes on weather conditions. We started by randomly generating 1500 sets of latitude and longitude coordinates. These coordinates were run through citypy within python to identify the nearest city. These 1500 pairs of coordinates equated to 550+ unique cities around the world.

Armed with a list of cities, OpenWeatherMap API was utilized to pull the current weather conditions of the cities within our list. The returned API data was promptly loaded a dataframe. This allowed for an easy rendering of specific weather characteristics with respect to latitude with matplotlib. The weather characteristics investigated were: max temperature, humidity, wind speed, and cloudiness.
